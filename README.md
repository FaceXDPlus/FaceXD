# FaceXD

这是基于Live2D SDK和ARKit的面部追踪APP。  
由于FaceXD开发者账号被封，目前项目更新全部提交到AMGFace分支。请前往该分支下载最新源码。  

# 现在可以实现的功能

- [x] 让Live2D模型动起来
- [x] 使用ARKit追踪面部表情
